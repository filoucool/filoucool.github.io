This is a project I am currently working on. This robot is called the PCBOT 914 and it is very old. My goal is to revive this robot and make it fully autonomous using the intel realsense cameras and IR distance sensors so that it can move around the school by himself. I will also add a return to home function and a self-charge function. Later on, I will add a mapping function using a Lidar scammer and will make an AI to detect humains and just say Hello for now... Eventually, I would like the robot to recognize all the students using OpenCV or tensorflow and I would like to make a small neural conversation model so that the robot can interact with students using IBM watson for voice!
